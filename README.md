# Amazon-Clone-01
A frontend clone of E-commerce website of Amazon. This is created using HTML,CSS and  Javascript. The page is created for big screens i.e, laptops, PCs only.

Project Done By:- Himanshu Shah 

The Amazon Clone project is a front-end e-commerce web application developed using HTML, CSS, and JavaScript. The main objective of this project is to recreate the design and basic functionality of the Amazon website while improving web development skills and understanding user interface design. This project includes various sections such as a responsive navigation bar, search bar, product categories, product listings, shopping cart interface, and footer section, similar to the original Amazon platform.

The website is designed using HTML for the structure, CSS for styling and responsive layouts, and JavaScript for adding interactivity and dynamic features. Users can browse products, view product details, and interact with elements like buttons, menus, and cart options. The responsive design ensures that the website works smoothly on different devices such as desktops, tablets, and mobile phones.

This project helps developers gain practical experience in front-end development concepts including responsive web design, DOM manipulation, event handling, layout management, and user-friendly interface creation. It also improves problem-solving and coding skills by implementing real-world e-commerce features. Overall, the Amazon Clone project is a beginner-friendly and practical application that demonstrates the use of modern web technologies to create an interactive online shopping experience.
